# Manual QA Test Plan – Order Fulfillment System

## Overview
This project demonstrates manual QA testing for an order fulfillment system. It includes test cases based on core workflow steps, along with a sample bug report to show structured defect documentation.

## Workflow Covered
The test plan focuses on the main order fulfillment flow:
- Order placement
- Inventory verification
- Warehouse item picking
- Shipping label generation
- Tracking number creation

## Contents
- Manual test cases for key workflow steps
- Positive and negative test coverage
- Sample bug report with reproduction steps, expected results, and actual results

## Tools & Skills Demonstrated
- Manual Testing
- Test Case Design
- Defect Documentation
- Workflow Validation
- Analytical Thinking
- QA Documentation

## Goal
The goal of this project is to demonstrate the ability to test a business process end-to-end, identify defects, and clearly document testing results in a structured and professional format.
